The file 0-print_listint.c contains a function that prints all the elements of a listint_t list.

The file 1-listint_len.c contains a function that returns the number of elements in a linked listint_t list.

The file 2-add_nodeint.c contains a function that adds a new node at the beginning of a listint_t list.

The file 3-add_nodeint_end.c a function that adds a new node at the end of a listint_t list.

The file 4-free_listint.c contains  a function that frees a listint_t list.

The file 5-free_listint2.c a function that frees a listint_t list.

The file 6-pop_listint.c contains a function that deletes the head node of a listint_t linked list,
and returns the head node’s data (n).

The file 7-get_nodeint.c contains a function that returns the nth node of a listint_t linked list.
