The file 0-read_textfile.c contains a function that reads a text file and prints it to the POSIX standard output.

The file 1-create_file.c Create a function that creates a file.

The file 2-append_text_to_file.c contains a function that appends text at the end of a file.

The file 3-cp.c  contains a program that copies the content of a file to another file.

The file 100-elf_header.c contains a program that displays the information contained in the ELF header at the start of an ELF file.
